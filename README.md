# PLC-NO-NC-Latching-Rung
This project demonstrates PLC ladder logic using Normally Open (NO) and Normally Closed (NC) push buttons with a latching circuit in a rung. Pressing the start button latches the output ON, while the stop button breaks the circuit and turns the output OFF. It shows basic PLC control used in industrial automation.
# PLC NO/NC Button with Latching Circuit

This project demonstrates PLC ladder logic using Normally Open (NO) and Normally Closed (NC) push buttons with a latching circuit.

## Working Principle
- The NO push button acts as the **Start button**.
- The NC push button acts as the **Stop button**.
- When the Start button is pressed, the output coil turns ON.
- A parallel contact creates a **latching circuit**, keeping the output ON even after releasing the Start button.
- Pressing the Stop button breaks the rung and turns the output OFF.

## Concept Used
- PLC Ladder Logic
- Normally Open (NO) Contact
- Normally Closed (NC) Contact
- Latching Circuit
- Industrial Automation Control
